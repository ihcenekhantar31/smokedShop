# Multilingual E-commerce Website Demo

A responsive multilingual e-commerce website demo built with React, Vite, Tailwind CSS, React Router, and i18next.

Live demo: https://smoked-shop.vercel.app

## Project Overview

This project is a demo e-commerce website for a seafood shop. It includes product pages, product categories, cart functionality, delivery information, responsive design, and multilingual support.

Private business/client information was replaced with demo content for confidentiality.

## Technologies Used

- React
- Vite
- Tailwind CSS
- React Router
- i18next
- JavaScript
- Vercel

## Features

- Responsive design for desktop and mobile
- Multilingual website
- Product catalog
- Product pages
- Shopping cart structure
- Delivery information page
- Clean UI for a local business
- Deployment with Vercel
